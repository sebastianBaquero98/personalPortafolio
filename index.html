My Projects

Ice Breaker

Icebreaker is a random pickup-line generator. 3 phrases will be generated and the user will have the option to use the pickup-line or not. If the user decides to use it he will win points. His score will be saved in his profile. He will be able to share his score with his friends. This web app was made using django framework.

Moviez

Moviez is a mobile application which recomends movies to the user. The app was heavily inspired in the Tinder app which uses a swipe mecanism. The user can swipe through movies indicating if he likes the movie or not. The recomendation algorithm will use this data to improve the recomendation that it gives. The app was developed using flutter a framework that permits you write code once and compile code for both IOS and android.
